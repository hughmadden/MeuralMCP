# Agent Notes

This repository is intended to be public. Keep it generic:

- Do not commit real Meural device names, cloud IDs, serials, MAC addresses, or IP addresses.
- Do not commit credentials, API tokens, cookies, `.env` files, generated state, or private images.
- Use examples such as `canvas-1`, `canvas-2`, and `192.0.2.x` if documentation needs sample identifiers.
- Keep runtime secrets in local env/config files ignored by git.
- Prefer generic sample config in `examples/`.
